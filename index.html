Overview

claude.ai/design이란?

Anthropic이 개발한 AI 기반 디자인 도구입니다. 자연어 프롬프트만으로 웹 UI, 프레젠테이션, 인터랙티브 프로토타입, 애니메이션 등 다양한 디자인 결과물을 즉시 생성합니다.

🔗 URL claude.ai/design
🆓 요금 Claude 구독 포함
Features

주요 기능 한눈에 보기

  • 자연어 프롬프트만으로 디자인 생성
  • 실제로 동작하는 인터랙티브 프로토타입
  • PPTX · PDF · HTML · Canva 등 다양한 내보내기
  • 디자인 시스템 설정 및 자동 적용
  • Claude Desktop · CLI · MCP 연동
How it works

작동 원리

Claude가 HTML/CSS/JS를 직접 작성하여 브라우저 내에서 즉시 렌더링합니다. 결과물은 단순 이미지가 아니라 실제 동작하는 코드입니다.

팁: "수정해줘"라는 후속 프롬프트로 결과물을 반복 개선할 수 있습니다.

Context

컨텍스트와 함께 시작

브랜드 가이드, 레퍼런스 이미지, 기존 디자인 파일을 첨부하면 더 정확한 결과물을 생성합니다.

프롬프트 예시

"첨부한 브랜드 가이드 기반으로 SaaS 랜딩페이지 히어로 섹션을 만들어줘. 파란색 계열, 간결한 카피."

프롬프트 예시

"SaaS 서비스 랜딩페이지 처음부터 만들어줘. 히어로, 기능 소개, CTA 섹션 포함."

Interactive

Interactive Prototype

실제로 클릭하고 동작하는 인터랙티브 프로토타입을 생성합니다. 버튼, 메뉴, 폼 등 실제 사용자 흐름을 즉시 테스트할 수 있습니다.

프롬프트 예시

"모바일 앱 온보딩 플로우 3단계 인터랙티브 프로토타입 만들어줘."

"쇼핑몰 상품 상세 페이지 — 이미지 갤러리 클릭, 수량 선택, 장바구니 담기 동작 구현."

"SaaS 사이드바 네비게이션 — 메뉴 접기/펼치기, 활성 상태 표시 인터랙션 포함."

Presentation

Make a Deck

HTML 기반 슬라이드 프레젠테이션을 생성합니다. 키보드 방향키로 넘기는 실제 동작하는 덱을 만들어줍니다.

프롬프트 예시

"스타트업 투자자용 피치덱 10장. 문제 정의부터 팀 소개까지."

"분기 실적 보고서 덱. 매출 차트 3개 포함, 깔끔한 기업 스타일."

"신입 직원 온보딩 자료 덱. 회사 소개, 프로세스, 팀 구성 순서로."

Animation

Animated Video

타임라인 기반 모션 디자인을 생성합니다. CSS 애니메이션과 JS를 활용한 움직이는 디자인 결과물을 만들 수 있습니다.

프롬프트 예시

"로고 리빌 애니메이션 만들어줘. 페이드인 후 슬라이드업 효과."

"통계 숫자 카운팅 애니메이션. 0에서 목표 수치까지 2초에 걸쳐 증가."

"제품 기능 3가지를 순서대로 소개하는 슬라이드쇼 애니메이션."

Customization

Make Tweakable

디자인 내에 트윅 컨트롤(색상 피커, 슬라이더 등)을 추가합니다. 코드 없이도 디자인 변수를 실시간으로 조정할 수 있습니다.

프롬프트 예시

"색상과 폰트 크기를 조정할 수 있는 트윅 패널 추가해줘."

"히어로 섹션 배경색·타이틀·서브타이틀을 모두 실시간 편집 가능하게 해줘."

"카드 컴포넌트에 라운드, 그림자, 패딩 조절 슬라이더 컨트롤 추가해줘."

UI Design

Frontend Design

기존 브랜드 시스템 없이도 미적으로 완성도 높은 UI 디자인을 생성합니다. Claude가 색상, 타이포그래피, 레이아웃을 직접 결정합니다.

프롬프트 예시

"모던하고 미니멀한 SaaS 대시보드 홈 화면 디자인해줘."

"여행 앱 메인 화면. 따뜻한 톤, 대형 사진 중심 레이아웃."

"B2B 서비스 랜딩페이지. 신뢰감 있는 네이비 계열, 기능 3단 비교표 포함."

Exploration

Wireframe

아이디어를 빠르게 탐색하기 위한 와이어프레임과 스토리보드를 생성합니다. 여러 레이아웃 방향을 동시에 비교할 수 있습니다.

프롬프트 예시

"대시보드 레이아웃 와이어프레임 3가지 방향으로 보여줘."

"이커머스 체크아웃 플로우 와이어프레임. 장바구니→배송→결제 3단계."

"앱 IA 다이어그램. 주요 화면 5개와 화면 간 이동 흐름 표시."

내보내기

결과물을 다양한 형식으로 저장
PowerPoint

PPTX 내보내기 (편집 가능)

네이티브 텍스트와 도형으로 구성된 편집 가능한 PowerPoint 파일로 내보냅니다. PowerPoint에서 직접 수정할 수 있습니다.

.pptx 편집 가능 네이티브 요소
적합한 경우: 팀이 내용을 직접 수정해야 하는 내부 발표, 반복 사용하는 보고서 템플릿.
PowerPoint

PPTX 내보내기 (스크린샷)

픽셀 퍼펙트 스크린샷 이미지로 구성된 PowerPoint 파일로 내보냅니다. 편집은 불가하지만 디자인이 그대로 유지됩니다.

.pptx 픽셀 퍼펙트 읽기 전용
적합한 경우: 투자자·클라이언트 발표 — 디자인을 수정 없이 정확히 전달해야 할 때.
PDF

PDF로 저장

인쇄용 PDF로 내보냅니다. 프레젠테이션, 보고서, 포트폴리오 등 공유 목적으로 활용하기 좋습니다.

.pdf 인쇄 최적화 공유 적합
적합한 경우: 제안서, 포트폴리오, 인쇄물 — 어떤 기기·OS에서도 동일하게 보여야 할 때.
HTML

독립 HTML로 저장

외부 의존성 없이 오프라인에서도 동작하는 단일 HTML 파일로 내보냅니다. 인터랙티브 요소가 그대로 유지됩니다.

bash 
# 저장된 HTML 파일을 브라우저에서 실행
open design-export.html
적합한 경우: 인터랙티브 프로토타입 보존, 개발팀 핸드오프, 인터넷 없는 환경 시연.
Canva

Canva로 보내기

디자인을 편집 가능한 Canva 디자인으로 내보냅니다. Canva에서 추가 편집하거나 팀원과 협업할 수 있습니다.

Canva 연동 편집 가능 팀 협업
적합한 경우: 팀이 Canva를 주로 사용할 때, SNS 콘텐츠로 추가 편집이 필요할 때.
Claude Code

Claude Code로 인계

디자인이 완성되면 핸드오프 번들로 패키징하여 Claude Code에 전달합니다. 디자인 의도(design intent)가 포함되어 Claude Code가 프로덕션 코드를 자동 구현합니다.

핸드오프 번들 디자인 의도 포함 단방향
인계 옵션 3종: Claude Code로 인계 · 로컬 코딩 에이전트로 전송 · Claude Code Web으로 전송 중 선택.

Setup

디자인 시스템 설정

  1. 1설정 패널에서 "Design System" 선택
  2. 2브랜드 색상 팔레트 입력
  3. 3타이포그래피 규칙 설정
  4. 4컴포넌트 스타일 지정
Overview

디자인 시스템 개요

설정된 디자인 시스템은 색상 토큰, 타이포그래피 스케일, 간격 규칙 등을 포함합니다. Claude가 이를 참조하여 브랜드 가이드라인에 맞는 결과물을 생성합니다.

Usage

디자인 시스템 활용

디자인 시스템이 설정된 후에는 Claude가 자동으로 브랜드 색상과 폰트를 사용합니다. 별도로 지정하지 않아도 일관된 결과물이 나옵니다.

팁: "우리 브랜드 가이드라인대로"라고만 해도 시스템에 설정된 값이 자동 적용됩니다.
Status

공개 디자인 시스템 현황

현재 claude.ai/design에는 공개된(published) 디자인 시스템이 없습니다. 디자인 시스템 기능은 각 사용자가 직접 설정하여 비공개로 사용합니다.

참고: 팀·조직 내에서 공유하려면 Team 또는 Enterprise 플랜의 공유 설정을 활용하세요.

연동

Claude Desktop · CLI · MCP
Handoff

Claude Code 핸드오프

디자인이 완성되면 Claude가 모든 내용을 핸드오프 번들로 패키징합니다. 디자인 의도(design intent)까지 포함된 번들을 Claude Code에 전달하면 프로덕션 코드로 자동 구현됩니다.

  1. 1디자인 완성 후 오른쪽 상단 내보내기 버튼 클릭
  2. 2Claude Code로 인계 선택 (또는 로컬 코딩 에이전트 · Claude Code Web)
  3. 3Claude Code가 번들 수신 → 디자인 의도 기반 코드 구현
단방향 핸드오프: Design → Code 방향만 존재. Claude Code에서 claude.ai/design을 직접 제어하는 구조는 현재 없음.
CLI

Claude CLI 연동

터미널에서 Claude CLI를 통해 디자인 생성을 자동화할 수 있습니다. CI/CD 파이프라인과 통합하거나 배치 작업에 활용합니다.

bash 
# Claude CLI 설치
npm install -g @anthropic-ai/claude-code

# 디자인 생성 요청
claude "랜딩페이지 히어로 섹션 디자인해줘"
MCP — 미지원

MCP 연동 — 향후 지원 예정

현재 claude.ai/design은 MCP(Model Context Protocol) 서버 연동을 지원하지 않습니다. Claude 채팅창에서 MCP를 통해 Claude Design을 제어하는 구조는 현재 없습니다.

향후 지원 예정: Anthropic 공식 발표에서 "향후 몇 주 내 통합 강화"를 예고했습니다. MCP 공식 지원 시 이 가이드에 업데이트됩니다.
MCP Servers

주요 MCP 서버 목록

  • Filesystem — 로컬 파일 읽기/쓰기 접근
  • Figma — Figma 파일 및 컴포넌트 가져오기
  • Notion — Notion 페이지 콘텐츠 연동
  • GitHub — 레포지토리 파일 직접 접근
  • Brave Search — 웹 검색 결과 활용
참고: 이 목록은 Claude Desktop 앱에서 사용하는 MCP 서버입니다. claude.ai/design 전용 MCP는 현재 미지원. 전체 목록은 modelcontextprotocol.io 참조.
Install

Claude Desktop 설치

Claude Desktop은 Mac과 Windows를 지원하는 네이티브 앱입니다. 설치 후 MCP 서버 연동 및 로컬 파일 접근이 가능해집니다.

  1. 1claude.ai/download 에서 OS에 맞는 설치 파일 다운로드
  2. 2설치 후 Claude 계정으로 로그인
  3. 3Settings → Developer → MCP 서버 설정 추가
참고: MCP 설정 파일 위치 — Mac: ~/Library/Application Support/Claude/claude_desktop_config.json
